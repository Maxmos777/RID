# Biblioteca de Literatura PMO

Referências bibliográficas organizadas por domínio semântico para uso pelos agentes e skills do `ring-pmo-team`.

---

## Estrutura

```
docs/
├── standards/
│   ├── project-management/        ← Normas PMI de gestão de projecto
│   └── portfolio-strategy/        ← Normas PMI de portfólio e OPM
├── practice-guides/
│   ├── pmo-and-processes/         ← Guias operacionais de PMO e grupos de processo
│   └── organizational-change/     ← Gestão de mudança organizacional
├── business-analysis/             ← Análise de negócio (BA)
├── strategy-mapping/              ← Mapeamento estratégico (Wardley)
├── transformation/                ← Transformação organizacional e IA
├── frameworks-and-modern-delivery/← Frameworks complementares e entrega moderna
└── visual-references/
    ├── pmbok-process-flows/       ← Fluxos de processo PMBOK (Ricardo Vargas)
    └── simplified-process-posters/← Cartazes simplificados de grupos de processo
```
